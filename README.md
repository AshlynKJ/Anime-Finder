App Description: This app allows user to search for animes based on the genre. They can then add an anime based on title to a database with an associated rank. They are also able to see a table of all the animes that they have ranked.

YouTube Video Demo Link: https://www.youtube.com/watch?v=cqSoS9mY6Xg

APIs: Kitsu API (https://kitsu.docs.apiary.io/#introduction/json:api)
